Featured Analytical Projects

Project One | Health & Social Data

Global HPV Virus Rampage Analysis

This project involved a rigorous analysis of global public health data to identify the correlation between HPV virus prevalence and key socio-economic factors. I led the initial data preparation in **Excel** (using VLOOKUPs and Pivot Tables) and built a comprehensive **Power BI** model, developing DAX measures to calculate prevalence rates. The resulting dashboard offered actionable insights by visually breaking down case rates against social determinants like continental distribution and economic status, forming a data-backed foundation for intervention strategies.

Key Features:

  • Power BI Visualization: Developed an interactive dashboard to explore case rates, immunization progress, and mortality.
  • Social Factor Breakdown: Analyzed prevalence against social factors like continental distribution, economic status, and access to education.
  • Excel Data Modeling: Used advanced functions for initial cleaning, segmentation, and preparation before loading into Power BI.

Project Two | Collaborative Web Data

African University Data Compilation & Web Development

**Challenge:** The objective was to move beyond varied reporting standards to compile, harmonize, and standardize key data from over 100 African universities, then present this complex information via a public-facing website. I worked directly with the Student Information Systems (SIS) team, defining the schema and logic for the vast dataset (Data Harmonization/ETL) and defining the API logic for the web display. The resulting public website became a centralized, objective resource, improving decision-making quality for students and policymakers.

Key Features:

  • Data Harmonization & ETL: Defined the schema and categorization rules for courses and regional demographics to ensure data consistency.
  • Metric Development: Defined KPIs like Student Demographics (tracking enrollment) and Research Output Mapping for showcasing strengths.
  • Cross-Functional Handoff: Provided the SIS development team with the final, validated data structure and necessary API logic.
